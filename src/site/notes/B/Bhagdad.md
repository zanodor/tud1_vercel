# Bhagdad

Iraki főváros neve, jelentése Isten-adta. Előtagja [[B/BOG\|BOG]], [[B/BOK\|BOK]] = Nap (szláv nyelvekben `bog`, `bózse` már maga Isten), utótagja [[A/Ad\|ad]] szavunk (mely megfordítottan – is – létezik utódnyelveinkben).
