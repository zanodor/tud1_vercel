# Bögre

Kétszer is megvan benne a Nap neve, de [[B/BÖG\|BÖG]] itt tellurikus, mert minden tároló, edény, [[K/Kehely\|kehely]] nőiségi kell legyen. [[T/Teknő\|Teknő]] is hasonlóan tárolóedény.
