# Bögre

Kétszer is megvan benne a Nap neve, de [[B/BÖG\|BÖG]] itt telluris, mert minden tároló, edény, [[K/Kehely\|kehely]] nőiségi kell legyen. [[T/Teknő\|Teknő]] is hasonlóan tárolóedény.
