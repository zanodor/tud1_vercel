# Corrie

Kör alakú vájat domb, hegy oldalában, skót angolságban (lásd még általánosabb [[C/Cirque\|cirque]]). Kör, kerek, karika szavaink láthatók benne. Lásd még (walesi-angol) [[C/Cwm\|cwm]].
