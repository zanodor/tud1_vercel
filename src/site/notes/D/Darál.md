# Darál

[[D/Deer\|Deer]] címnél is előjött darál szavunk:  
Darál szavunk is a [[H/Hamlet malma\|Hamlet malma]] forgására, őrlésére utal, forgómozgásra (mert ugye az [[E/Életfa\|életfa]], mint Világtengely forog).
